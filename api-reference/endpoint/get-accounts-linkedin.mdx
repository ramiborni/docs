Retrieve a list of all LinkedIn accounts that have been connected to your SendPilot project.

## Response

<ResponseField name="accounts" type="array" required>
  Array of connected LinkedIn accounts.
</ResponseField>

<ResponseField name="total" type="number" required>
  Total number of accounts.
</ResponseField>

### Account Object Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | string | ✅ | LinkedIn account ID |
| `status` | string | ✅ | Account status (see status values below) |
| `firstName` | string | | First name |
| `lastName` | string | | Last name |
| `email` | string | | Email address |
| `headline` | string | | Professional headline |
| `location` | string | | Location |
| `profilePictureUrl` | string | | Profile picture URL |
| `publicProfileUrl` | string | | Public LinkedIn profile URL |
| `publicIdentifier` | string | | Public identifier (username) |
| `premium` | boolean | ✅ | Whether account has LinkedIn Premium |
| `openProfile` | boolean | ✅ | Whether profile is open |
| `isConnected` | boolean | ✅ | Connection status |
| `createdAt` | string | ✅ | When account was connected |
| `updatedAt` | string | ✅ | Last update timestamp |

## Account Status Values

| Status | Description | Can Send Messages |
|--------|-------------|-------------------|
| `CONNECTED` | Account is connected and operational | ✅ Yes |
| `DISCONNECTED` | Account session has expired or been invalidated | ❌ No |
| `PENDING_VERIFICATION` | Awaiting verification | ❌ No |
| `TWO_FA_REQUIRED` | Two-factor authentication required | ❌ No |
| `OTP_REQUIRED` | One-time password required | ❌ No |
| `CAPTCHA_REQUIRED` | CAPTCHA challenge required | ❌ No |
| `IN_APP_VALIDATION_REQUIRED` | In-app validation required | ❌ No |
| `PHONE_REGISTER_REQUIRED` | Phone registration required | ❌ No |
| `FAILED` | Connection failed | ❌ No |
| `EXPIRED` | Session expired | ❌ No |

## Implementation Example

```javascript
const axios = require('axios');

async function listLinkedInAccounts() {
  const url = 'https://integration-api-gateway.agreeablesea-e4815929.swedencentral.azurecontainerapps.io/api/linkedin/accounts/linkedin';

  const response = await axios.get(url, {
    headers: {
      'X-API-KEY': process.env.SENDPILOT_API_KEY
    }
  });

  return response.data;
}

// Get only connected accounts
async function getConnectedAccounts() {
  const { accounts } = await listLinkedInAccounts();
  return accounts.filter(account => account.status === 'CONNECTED');
}

// Check account health
async function checkAccountHealth() {
  const { accounts, total } = await listLinkedInAccounts();
  
  const connectedCount = accounts.filter(a => a.status === 'CONNECTED').length;
  const disconnectedCount = accounts.filter(a => a.status === 'DISCONNECTED').length;
  
  console.log(`Total accounts: ${total}`);
  console.log(`Connected: ${connectedCount}`);
  console.log(`Disconnected: ${disconnectedCount}`);
  
  return { connectedCount, disconnectedCount, total };
}
```

- Set up alerts for accounts that become disconnected
